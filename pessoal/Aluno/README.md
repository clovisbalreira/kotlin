# Aluno

📌 **Descrição**

Aplicativo Android desenvolvido em Kotlin que calcula a média de notas de um aluno e determina sua situação (Aprovado/Reprovado).

🚀 **Tecnologias**

- Kotlin
- Android (AppCompatActivity)
- XML Layout
- JVM

🎯 **Funcionalidades**

- Entrada de nome do aluno
- Entrada de duas notas
- Cálculo automático da média
- Verificação de situação (aprovado com média >= 6)
- Exibição formatada dos dados e resultado

▶️ **Como rodar**

```bash
# Navegar até o diretório do projeto
cd pessoal/Aluno

# Compilar e executar com Gradle
./gradlew build

# Windows
gradlew.bat build

# Para executar em um emulador ou dispositivo Android
./gradlew installDebug
```
