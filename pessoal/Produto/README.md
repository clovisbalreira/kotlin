# Produto

📌 **Descrição**

Aplicativo Android desenvolvido em Kotlin para gerenciar informações de produtos, incluindo nome, descrição, valor unitário e quantidade.

🚀 **Tecnologias**

- Kotlin
- Android (AppCompatActivity)
- XML Layout
- Classes modelo
- JVM

🎯 **Funcionalidades**

- Cadastro de produtos com nome e descrição
- Entrada de valor unitário e quantidade
- Criação de objetos da classe Produto
- Cálculo de valor total (unitário × quantidade)
- Exibição formatada dos dados do produto
- Conversão automática de tipos

▶️ **Como rodar**

```bash
# Navegar até o diretório do projeto
cd pessoal/Produto

# Compilar e executar com Gradle
./gradlew build

# Windows
gradlew.bat build

# Para executar em um emulador ou dispositivo Android
./gradlew installDebug
```

- Criação e uso de classes modelo
- Manipulação de EditText e TextView
- Conversão de String para Double e Int
- Formatação de saída com interpolação
- Encapsulamento de lógica em classes
