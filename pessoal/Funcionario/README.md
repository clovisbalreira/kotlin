# Funcionário

📌 **Descrição**

Aplicativo Android desenvolvido em Kotlin para gerenciar dados de funcionários, capturando informações pessoais e profissionais com navegação entre telas.

🚀 **Tecnologias**

- Kotlin
- Android (AppCompatActivity)
- XML Layout
- Intent e Bundle
- JVM

🎯 **Funcionalidades**

- Entrada de dados do funcionário (nome, CPF, chapa, idade)
- Entrada de informações salariais (valor/hora, carga horária mensal)
- Uso de Intent para navegação entre Activities
- Passar dados entre telas usando Bundle
- Classe modelo Funcionario com atributos tipados
- Tela de visualização dos dados cadastrados

▶️ **Como rodar**

```bash
# Navegar até o diretório do projeto
cd pessoal/Funcionario

# Compilar e executar com Gradle
./gradlew build

# Windows
gradlew.bat build

# Para executar em um emulador ou dispositivo Android
./gradlew installDebug
```

📚 **Aprendizados**

- Desenvolvimento de múltiplas Activities em Android
- Uso de Intent para navegação entre telas
- Passagem de dados entre Activities com Bundle
- Criação de classes modelo
- Diferentes tipos primitivos (Long, Int, Byte, Float, Double)
- Conversão de tipos em aplicações Android
- Estrutura MVC em Android
