# Soma

📌 **Descrição**

Aplicativo Android simples desenvolvido em Kotlin que realiza a soma de dois números inseridos pelo usuário.

🚀 **Tecnologias**

- Kotlin
- Android (AppCompatActivity)
- XML Layout
- JVM

🎯 **Funcionalidades**

- Entrada de dois números decimais
- Cálculo da soma
- Exibição do resultado formatado
- Interface simples e intuitiva
- Event listener para o botão de cálculo

▶️ **Como rodar**

```bash
# Navegar até o diretório do projeto
cd pessoal/soma

# Compilar e executar com Gradle
./gradlew build

# Windows
gradlew.bat build

# Para executar em um emulador ou dispositivo Android
./gradlew installDebug
```
