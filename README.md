# Repositórios de Linguagem
## Exercícios em Kotlin

📌 **Descrição**

Repositório com exercícios e projetos feitos em Kotlin, desde conceitos básicos até aplicações mais complexas, seguindo a progressão do curso Estudonauta.

🧠 **Conteúdo**

- Lógica de programação
- Tipos primitivos e variáveis
- Operadores (aritméticos, relacionais, lógicos)
- Estruturas de controle
- Funções e métodos
- Manipulação de dados e strings
- Entrada e saída de dados
- Datas e horas
- Valores aleatórios
- Projetos pessoais (Aluno, Funcionário, Produto)

▶️ **Como rodar**

**Pré-requisitos:**
- Kotlin instalado
- Gradle configurado (já incluído no projeto)

**Executar um projeto:**

```bash
# Navegar até o diretório do projeto
cd estudonauta/000_Ola_Mundo

# Compilar e executar
./gradlew run

# No Windows
gradlew.bat run
```

**Compilar um projeto específico:**

```bash
./gradlew build
```

**Executar testes:**

```bash
./gradlew test
```

📝 **Notas**

- Cada pasta é um projeto Gradle independente
- Os exercícios são numerados em ordem crescente de dificuldade
- O projeto utiliza Gradle como gerenciador de dependências
- Estrutura ideal para aprendizado progressivo de Kotlin

🚀 **Tecnologias**

- Kotlin
- Gradle
- JVM
