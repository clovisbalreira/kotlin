# Variáveis e Constantes

📌 **Descrição**

Entenda a diferença entre variáveis (var) e constantes (val) em Kotlin e quando usar cada uma.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Declaração de constantes com val
- Variáveis mutáveis com var
- Imutabilidade em Kotlin
- Diferenças práticas

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 005_VariaveisConstantes

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Diferença entre val e var
- Benefícios da imutabilidade
- Escolha correta entre constantes e variáveis
- Segurança e boas práticas
