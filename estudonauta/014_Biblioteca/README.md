# Biblioteca

📌 **Descrição**

Aprenda a usar bibliotecas e dependências externas em projetos Kotlin com Gradle.

🚀 **Tecnologias**

- Kotlin
- JVM
- Gradle

🎯 **Funcionalidades**

- Importação de bibliotecas
- Uso de dependências
- Configuração de gradle
- Bibliotecas populares

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 014_Biblioteca

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Como importar e usar bibliotecas
- Configuração de dependências no Gradle
- Exploração de APIs externas
- Reutilização de código
