# Comentários

📌 **Descrição**

Estude como usar comentários em seu código Kotlin. Saiba a diferença entre comentários de linha única e múltiplas linhas.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Comentários de linha única (//)
- Comentários de múltiplas linhas (/* */)
- Boas práticas de documentação

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 002_Comentarios

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Tipos de comentários em Kotlin
- Quando e como usar comentários
- Melhor legibilidade do código
