# Data e Hora

📌 **Descrição**

Trabalhe com datas e horas em Kotlin usando LocalDateTime e explore manipulação temporal.

🚀 **Tecnologias**

- Kotlin
- JVM
- java.time

🎯 **Funcionalidades**

- LocalDateTime para data e hora
- Extração de componentes (dia, mês, hora)
- Manipulação de datas
- Formatação de data/hora

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 020_Data_Hora

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- API java.time do Kotlin
- LocalDateTime e seus métodos
- Manipulação de datas temporais
- Formatação de saída de data/hora
