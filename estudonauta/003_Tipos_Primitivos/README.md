# Tipos Primitivos

📌 **Descrição**

Explore todos os tipos de dados primitivos disponíveis em Kotlin: números inteiros, reais, caracteres e booleanos.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Tipos inteiros (Byte, Short, Int, Long)
- Tipos reais (Float, Double)
- Caracteres e Strings
- Booleans
- Tipo Any

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 003_Tipos_Primitivos

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Identificação e uso de tipos primitivos
- Ranges de valores para cada tipo
- Conversão entre tipos
- Type inference em Kotlin
