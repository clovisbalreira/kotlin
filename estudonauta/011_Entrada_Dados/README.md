# Entrada de Dados

📌 **Descrição**

Aprenda a receber dados do usuário através do console usando readLine() e outras técnicas de input em Kotlin.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Leitura com readLine()
- String interpolation com dados
- Conversão de tipos na entrada
- Tratamento de dados do usuário

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 011_Entrada_Dados

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Função readLine() do Kotlin
- Leitura de entrada padrão
- Conversão de strings para outros tipos
- Interação com usuário no console
