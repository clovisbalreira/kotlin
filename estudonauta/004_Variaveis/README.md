# Variáveis

📌 **Descrição**

Aprenda a declarar, inicializar e utilizar variáveis em Kotlin. Entenda as diferenças entre var e val.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Declaração de variáveis com var
- Type inference automático
- Atribuição de valores
- Escopo de variáveis

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 004_Variaveis

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Declaração e inicialização de variáveis
- Tipagem em Kotlin
- Nomeação de variáveis
- Boas práticas de declaração
