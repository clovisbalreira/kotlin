# Entrada com Validação

📌 **Descrição**

Implemente técnicas para validar e verificar dados de entrada do usuário antes de processar.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Validação de tipos
- Verificação de valores
- Try-catch para erros
- Loops de validação

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 012_Entrada_Validando

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Tratamento de erros de conversão
- Validação de entrada
- Try-catch em Kotlin
- Garantir integridade dos dados
