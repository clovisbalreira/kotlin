# Analisando Tipos

📌 **Descrição**

Explore técnicas para analisar e verificar tipos de variáveis em tempo de execução usando Kotlin.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Verificação de tipos com is
- Type checking e type casting
- Função javaClass
- Análise dinâmica de tipos

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 007_Analisando_Tipos

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Verificação de tipos em runtime
- Smart cast em Kotlin
- Casting explícito e seguro
- Análise de tipos dinâmicos
