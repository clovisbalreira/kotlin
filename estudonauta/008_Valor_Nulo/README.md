# Valor Nulo (Null Safety)

📌 **Descrição**

Entenda o sistema de null safety do Kotlin e como trabalhar com valores nulos de forma segura.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Tipos nullable e non-null
- Operador ?
- Elvis operator ?:
- Safe call operator ?.

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 008_Valor_Nulo

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Nullability em Kotlin
- Prevenção de NullPointerException
- Operadores de null safety
- Boas práticas com valores nulos
