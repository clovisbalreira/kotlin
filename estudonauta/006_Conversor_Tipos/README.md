# Conversor de Tipos

📌 **Descrição**

Aprenda a converter valores entre diferentes tipos primitivos em Kotlin usando funções de conversão.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Conversão de inteiros para decimais
- Conversão de Float para Double
- Métodos de conversão (toInt, toFloat, etc)
- Explicitação de tipos

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 006_Conversor_Tipos

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Funções de conversão em Kotlin
- Perda de precisão em conversões
- Type casting seguro
- Compatibilidade entre tipos
