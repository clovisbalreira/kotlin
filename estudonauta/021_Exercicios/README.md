# Exercícios Práticos

📌 **Descrição**

Coleção de exercícios práticos que consolidam todos os conceitos aprendidos nos módulos anteriores de Kotlin.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Exercícios de lógica
- Problemas de programação
- Desafios incrementais
- Aplicação de conceitos

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 021_Exercicios

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Consolidação de conceitos
- Prática e reforço
- Resolução de problemas
- Pensamento algorítmico
