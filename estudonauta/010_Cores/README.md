# Cores

📌 **Descrição**

Aprenda a trabalhar com cores no console usando códigos ANSI e técnicas de colorização de texto em Kotlin.

🚀 **Tecnologias**

- Kotlin
- JVM
- ANSI Escape Codes

🎯 **Funcionalidades**

- Cores de texto
- Cores de fundo
- Estilos (negrito, itálico)
- Formatação colorida

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 010_Cores

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Códigos ANSI para cores
- Colorização de console
- Formatação visual em output
- Melhor apresentação de dados
