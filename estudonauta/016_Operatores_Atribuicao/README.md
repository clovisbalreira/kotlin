# Operadores de Atribuição

📌 **Descrição**

Aprenda os diferentes operadores de atribuição em Kotlin para modificar valores de variáveis.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Atribuição básica (=)
- Soma e atribuição (+=)
- Subtração e atribuição (-=)
- Multiplicação e atribuição (*=)
- Divisão e atribuição (/=)
- Módulo e atribuição (%=)

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 016_Operatores_Atribuicao

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Operadores de atribuição composta
- Atalhos para operações comuns
- Sintaxe simplificada
- Código mais conciso
