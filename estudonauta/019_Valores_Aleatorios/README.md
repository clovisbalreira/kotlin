# Valores Aleatórios

📌 **Descrição**

Aprenda a gerar números aleatórios em Kotlin usando Random e explore suas funcionalidades.

🚀 **Tecnologias**

- Kotlin
- JVM
- kotlin.random.Random

🎯 **Funcionalidades**

- Geração de números aleatórios
- Números em intervalo
- Diferentes tipos de dados
- Seeds para reprodutibilidade

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 019_Valores_Aleatorios

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Classe Random do Kotlin
- Geração de aleatoriedade
- Intervalos de valores
- Aplicações práticas
