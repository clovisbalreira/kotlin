# Operadores Relacionais

📌 **Descrição**

Estude operadores relacionais para comparação de valores: igual, diferente, maior, menor, etc.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Igual (==)
- Diferente (!=)
- Maior (>)
- Menor (<)
- Maior ou igual (>=)
- Menor ou igual (<=)

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 015_Operatores_relacionais

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Operadores de comparação
- Valores booleanos
- Uso em estruturas condicionais
- Comparação de tipos diferentes
