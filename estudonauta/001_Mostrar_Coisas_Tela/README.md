# Mostrar Coisas na Tela

📌 **Descrição**

Aprenda diferentes formas de exibir informações e dados na tela usando Kotlin. Explora vários métodos de output no console.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Comandos de saída (println, print)
- Formatação de texto
- Múltiplas formas de exibição

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 001_Mostrar_Coisas_Tela

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Diferença entre println e print
- Exibição formatada de dados
- Técnicas básicas de output no console
