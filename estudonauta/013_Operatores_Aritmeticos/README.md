# Operadores Aritméticos

📌 **Descrição**

Explore todos os operadores aritméticos disponíveis em Kotlin: adição, subtração, multiplicação, divisão e módulo.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Adição (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/)
- Módulo (%)

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 013_Operatores_Aritmeticos

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Operadores matemáticos básicos
- Precedência de operações
- Divisão inteira vs decimal
- Classe Math do Kotlin
