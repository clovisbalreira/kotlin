# Olá Mundo

📌 **Descrição**

Primeiro programa em Kotlin! Um simples "Hello World" que demonstra como criar e executar seu primeiro aplicativo na linguagem.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Exibição de mensagem no console
- Função main como ponto de entrada
- Comando println para saída

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 000_Ola_Mundo

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Estrutura básica de um programa Kotlin
- Função main e seus parâmetros
- Comando println para imprimir mensagens
- Conceitos fundamentais da linguagem
