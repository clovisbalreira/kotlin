# Operadores Lógicos

📌 **Descrição**

Dominе operadores lógicos em Kotlin: AND (&&), OR (||) e NOT (!) para criar expressões booleanas complexas.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- E lógico (&&)
- OU lógico (||)
- NÃO lógico (!)
- Tabelas verdade
- Short-circuit evaluation

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 018_Operatores_Logicos

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Lógica booleana
- Combinação de condições
- Avaliação de curto-circuito
- Condições complexas
