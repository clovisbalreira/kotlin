# Saídas Interpoladas

📌 **Descrição**

Aprenda a usar string interpolation em Kotlin para criar strings dinâmicas e formatadas.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Interpolação com ${}
- Expressões em strings
- Template strings
- Formatação de saída

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 009_Saidas_Interpoladas

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- String interpolation em Kotlin
- Sintaxe de ${}
- Expressões complexas em strings
- Output formatado dinamicamente
