# Operadores de Incremento e Decremento

📌 **Descrição**

Explore operadores de incremento (++) e decremento (--) em Kotlin - versões pré e pós-fixadas.

🚀 **Tecnologias**

- Kotlin
- JVM

🎯 **Funcionalidades**

- Incremento pré-fixado (++var)
- Incremento pós-fixado (var++)
- Decremento pré-fixado (--var)
- Decremento pós-fixado (var--)
- Diferença entre pré e pós

▶️ **Como rodar**

```bash
# Navegar até o diretório
cd 017_Operatores_Incremento_Decremento

# Executar com Gradle
./gradlew run

# Windows
gradlew.bat run
```

📚 **Aprendizados**

- Diferença entre ++/-- pré e pós-fixado
- Retorno de valores
- Uso em loops
- Casos de uso práticos
